# SwITCh-v2
New version of SwITCh simulation.

# Coding conventions
 - Action and parameter
 ```gaml 
 action my_action(int my_param)
 ```
 - Attribute
 ```gaml
 float my_attribute
 ```
 - Reflex
 ```gaml
 reflex my_reflex
 ```
 - Species
 ```gaml
 species MySpecies
 ```
 - Model
 ```gaml
 model MyModel
 ```
 - Experiment
 ```gaml
 experiment "My experiment"
 ```
 - Display
 ```gaml
 display my_display
 ```
